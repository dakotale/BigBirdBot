# BigBirdBot
Discord Bot

BigBirdBot - Help

$ - Alias: $
Description: Ask the bot a question and receive a possibly accurate answer

-8ball - Alias: 8b
Description: Shake the figurative Eight Ball

-ascii - Alias: ascii
Description: Show Ascii of the message you type

-avatar - Alias: avatar
Description: Discord Profile Pic in HQ

-ban - Alias: ban
Description: Bans a user

-clear - Alias: clear
Description: Empties the queue

-equalizer - Alias: equalizer,eq
Description: Sets the audio EQ with three settings; Super bass, bass, and pop

-etext - Alias: etext,em
Description: Turn your message into emojis

-event - Alias: event,eve,plan
Description: Plan an event/notification in 'Date/Time, Event Name' format

-forceskip - Alias: forceskip,fs,fskip,skip
Description: Skip the current track playing

-help - Alias: help
Description: List of commands and descriptions available

-Join - Alias: join
Description: Joins the voice channel

-ka - Alias: ka
Description: Add your own keyword and action to the server. Example -ka laugh, LOL

-kaedit - Alias: kaedit
Description: Edit an existing keyword

-kanye - Alias: kanye
Description: Random tweet from Kanye's Twitter

-Leave - Alias: leave
Description: Leaves the voice channel

-loop - Alias: loop
Description: Loops the current track based on the number entered. Ex: -loop 5

-NowPlaying - Alias: nowplaying,np
Description: View the current playing track

-Pause - Alias: pause
Description: Pause the current audio playing

-Play - Alias: play
Description: Join and play audio from Youtube, Soundcloud, Spotify, or local files. Direct search -play "My Music"

-playnext - Alias: -playnext
Description: Places the track at the top of the queue

-poll - Alias: poll,p
Description: Create a poll with a maximum of 10 options

-queue - Alias: queue,q
Description: View the queue

-raffle - Alias: raffle
Description: Choose someone at random to win a prize

-random - Alias: random,r
Description: Random number out of a certain range.

-remove - Alias: remove
Description: Removes a specific element in the queue

-repeat - Alias: repeat
Description: Loops the current track once

-Resume - Alias: resume
Description: Resume the current audio paused

-shuffle - Alias: shuffle
Description: Randomizes the queue

-Stop - Alias: stop
Description: Stops the audio, clears the queue, and the bot will leave the voice channel.

-swap - Alias: swap
Description: Switches positions of two tracks in the queue

-trivia - Alias: trivia
Description: Random multiple choice question to answer

-Volume - Alias: volume,vol
Description: Set the volume between 0 and 150.

-weather - Alias: weather
Description: Look up the weather in 'City, State/Province, Country' format.

-wikip - Alias: wikip
Description: Random Wikipedia article
